# UniversalCrawl

UniversalCrawl is being recast as a planning-first effort to build the discovery layer for the agent internet: a system that observes the real action surface of websites so agents stop guessing at UI behavior and start reasoning over grounded capabilities.

This repository is intentionally sparse. The current phase is strategy, architecture, evaluation design, and risk analysis for a discovery-first platform; broad execution is explicitly deferred.
